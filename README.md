# Zigen IP-Logic Crestron Module
### Zigen.IPLogic
Crestron driver for Zigen IP-Logic AV Over IP Products.<br>

This module is written in SIMPL# with SIMPL/SIMPL+ wrapper for use in SIMPL Windows.
